# Manim_Documentation

This repository is made to help people that need help or some piece of documentation about a Python library called Manim and this guide will keep being filled with more info

## Goals

* 1st goal as I said is to help people "get to know Manim"
* 2nd goal is to inspire others to try building animations with Manim because it is a simple but usefull way to learn the Python language

## Repository Structure

* 00-getting-started
* 01-basics
* 02-intermediate
* 03-Troubleshooting
* assets

## Dependencies

* Python
* Manim
