# Let`s get to know Manim

- Manim is a python library used to create animations by coding.

- It was originally created by Grant Sanderson who runs a Youtube channel called 3Blue1Brown

There are 2 main versions, 
- ManimGL which is the version being developed by Grant Sanderson and the one he is using for his videos

- There`s also Manim Community Edition (ManimCE) and is maintained by the community and is more stable with easier access to documentation being the preferred for beginners.

- In this repo I am showing a full guide on Manim
